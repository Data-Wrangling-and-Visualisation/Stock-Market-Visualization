Russian Stock Returns Distribution

1. What is this visualization?
This visualization shows the distribution of returns for Russian stocks, combining a histogram (gray bars) and a density curve (blue line) to highlight how returns are distributed across companies.

2. Data used in this visualization
The chart uses performance data from companies listed on the Moscow Exchange (MOEX), showing either daily returns (recent market movement) or yearly returns (long-term performance trends).

3. Insights you can gain
With this chart, you can:

  • Identify the most common return ranges in the Russian market
  • Spot extreme performers (outliers) on both positive and negative sides
  • Understand if returns are evenly distributed or skewed toward gains/losses
  • Compare how short-term and long-term returns differ in their distribution patterns
  • Assess the overall market sentiment by seeing where most companies cluster

4. How to use the controls

  • Data Characteristic selector: Switch between daily and yearly returns to compare short and long-term patterns
  • Number of Bins slider: Adjust the granularity of the histogram - more bins show finer detail, fewer bins show broader trends
  • Hover functionality: Mouse over any part of the chart to see exact percentage values

Data source: Trading Economics - Russia Stock Market
